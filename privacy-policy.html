Политика обработки персональных данных

Индивидуального предпринимателя Гордеевой Юлии Владимировны

1. Общие положения

1.1. Настоящая Политика обработки персональных данных разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных Индивидуальным предпринимателем Гордеевой Юлией Владимировной (ИНН 501304589898, ОГРНИП 319774600004492) (далее — «Оператор»).

1.2. Политика действует в отношении всех персональных данных, которые Оператор получает от пользователей сайта hr.radarexec.ru (в дальнейшем — Сайт).

1.3. Использование Сайта означает согласие пользователя с настоящей Политикой.

2. Состав персональных данных

Оператор обрабатывает следующие данные, предоставляемые пользователем при обращении через Сайт: имя, номер телефона, адрес электронной почты, а также иные сведения, сообщённые пользователем добровольно.

3. Цели обработки

  • обработка заявок, поступивших через Сайт;
  • направление информационных и рекламных рассылок — при отдельном согласии пользователя;
  • статистический анализ посещаемости Сайта.

4. Способ сбора данных

Данные передаются пользователем напрямую на электронную почту Оператора (prof@jgordeeva.ru) посредством почтового клиента при отправке письма с заявкой или обращением через Сайт. Сторонние сервисы для приёма и обработки заявок Оператором не используются.

5. Передача третьим лицам

Для анализа посещаемости Сайта используется сервис «Яндекс.Метрика» (ООО «Яндекс»), которому передаются обезличенные данные о поведении пользователя на Сайте после получения согласия через cookie-баннер. Иным третьим лицам персональные данные не передаются. Трансграничная передача персональных данных не осуществляется.

6. Срок хранения

Персональные данные хранятся до отзыва согласия пользователем либо до достижения цели обработки.

7. Права пользователя

Пользователь вправе отозвать согласие на обработку персональных данных, запросить информацию об обработке своих данных, а также требовать их уточнения или удаления, направив письмо на адрес prof@jgordeeva.ru.
